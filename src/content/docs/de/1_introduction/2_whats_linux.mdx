Linux ist das am weitesten verbreitete Betriebssystem der Welt. Es ist kostenlos und quelloffen. Es wird auch für Android verwendet. Linux läuft auch hinter den Kulissen der meisten Websites. Man kann es aber auch auf einem PC benutzen.

## Warum nicht Windows?

- Windows kostet über 100 Euro und man merkt es nur nicht, weil die Gerätehersteller dafür bezahlen. Doch ein Computer ohne Windows kostet weniger.
- Windows ist langsam. Linux ist ein hochoptimiertes Betriebssystem, da Firmen ihre Server auf Linux laufen lassen. Selbst Microsoft verwendet Linux für seine Online-Dienste. Deshalb haben sie viel Geld in dieses System investiert. Amazon, Google, Meta und viele andere Unternehmen tun das Gleiche.
- Sie können nicht wissen, was Windows auf Ihrem Computer macht. Es ist ein geschlossenes System, dem man nicht trauen kann.
- Windows ist nicht sehr anpassungsfähig. Alle Lösungen, die es gibt, sind etwas unsauber.
- Windows ist sehr aufgebläht. Es werden viele veraltete und nutzlose Programme mitgeliefert.
- Es gibt keine offizielle Paketverwaltung. Die meiste Software muss in einem endlosen Konfigurationsdialog installiert werden und Updates müssen manchmal manuell durchgeführt werden.
- Microsoft spioniert dich aus. Sie verfolgen und verkaufen Deine Informationen.
- Windows aktualisiert sich selbst oder wechselt den Standardbrowser zu Microsoft Edge. Linux macht so etwas nie.
- Windows ist ein sehr altes Betriebssystem. Neue Versionen verwenden immer noch den gleichen Systemkern wie Windows 3.1!
- Windows 10 wird in ein paar Jahren auslaufen und Windows 11 wird auf den meisten älteren Geräten nicht mehr laufen. Du musst einen neuen Computer kaufen, Linux benutzen oder du könntest gehackt werden.

**Linux hat keines dieser Probleme! Es ist *kostenlos*, super *anpassungsfähig* und du kannst sogar den Quellcode lesen, wenn du willst!

## Doch was genau ist Linux?

Es ist der Kern des Betriebssystems. Es verwaltet die gesamte Hardware und die Prozesse. Es hat keine grafische Oberfläche.

## Desktop-Umgebungen

Es gibt verschiedene Softwarepakete, die eine grafische Oberfläche für Linux bieten. Sie werden als Desktop-Umgebungen bezeichnet.

## Gnome

Gnome ist eine sehr gut durchdachte und beliebte Desktop-Umgebung, die sich jedoch stark von Windows unterscheidet und für deren Anpassung Software von Drittanbietern installiert werden muss. Es ist sehr minimalistisch und hat standardmäßig keine Taskleiste.

### KDE Plasma (empfohlen)

KDE Plasma ist eine extrem anpassbare Desktop-Umgebung. Sie ist auch benutzerfreundlicher für Leute, die zuvor Windows verwendet haben. Es hat standardmäßig eine Taskleiste (wie Windows), die Panel genannt wird.

## Distributionen (kurz: Distros)

Um Linux benutzen zu können, braucht man eine Menge Software: Einen Paketmanager, ein Terminal, eine Desktop-Umgebung und noch einige andere Software. Außerdem gibt es immer wieder Diskussionen darüber, welche Software besser ist. Deswegen sind Distributionen so wichtig.

Stell dir eine Distribution wie eine Geschmacksrichtung von Linux vor. Sie ist vollgepackt mit Software, die du brauchst, um loszulegen. Sie konfiguriert auch alles für dich, damit alles gut zusammenarbeitet.

:::tip

Du solltest eine Distribution nicht nach ihrem Aussehen auswählen. Du solltest sie nach ihrem Paketmanager, ihrem Maintainer, ihrer Community und so weiter auswählen.

:::

### Debian

Debian ist die Mutter der meisten Linux-Distributionen, aber vielleicht **nicht die beste Wahl für Anfänger**. Außerdem ist es ziemlich alt.

### Ubuntu

Ubuntu ist eine **sehr beliebte und einsteigerfreundliche** Distribution, aber sie wird von einer Firma namens Canonical gepflegt. Und diese Firma trifft manchmal **Microsoft-artige** Entscheidungen. Zum Beispiel haben sie **eine winzige Werbung** in eine Anwendung namens "Terminal" eingebaut. Ubuntu basiert auf Debian und verwendet [Gnome](#gnome) als Desktop-Umgebung.

### Fedora (empfohlen)

Fedora ist eine sehr populäre Linux-Distribution. Es ist die **modernste** Linux-Distribution und sehr **einsteigerfreundlich**. Sie wird von Red Hat gesponsert, einer Firma, die auch eine auf Fedora basierende Unternehmensversion herstellt. Sie ist jedoch viel stärker auf die Gemeinschaft ausgerichtet als Ubuntu. Fedora ist mit vielen verschiedenen Desktopumgebungen verfügbar.

### Und noch mehr...

Arch, Manjaro, Kubuntu, Fedora Silverblue, Ubuntu Server, Endless OS, Zorin und viele mehr sind Linux-Distributionen.

## Fazit

Du könntest eine andere Distribution wählen, aber ich würde **Fedora mit KDE Plasma** empfehlen. Es ist super modern und trotzdem sehr einsteigerfreundlich. *Diese Anleitung bezieht sich auf Fedora mit KDE Plasma. Wenn Sie eine andere Distribution verwenden, kann es sein, dass einige Dinge nicht funktionieren oder anders sind. Wenn Sie eine andere Desktop-Umgebung verwenden möchten, kann es sein, dass einige Dinge anders aussehen oder anders funktionieren.
